## Install

```sh
git clone https://github.com/Col0ring/react-ts-quick-starter
```

## Usage

```sh
yarn
# or
npm install
```

## ✊ Features

- 基于 webpack 5 搭建，使用了 webpack5 自带的新特性
- 提供开箱即用的 react + typescript 开发环境
- 具有统一配置文件，可根据配置文件修改对应需求
- 具备完善的代码提交和检测功能
- 提供自带的 mock 数据功能，可接入`Mock.js`
